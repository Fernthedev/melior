pub type AstNode<'a> = Node<'a, RefCell<Ast>>;
Expand description

The type of a node within the document.

It is bound by the lifetime 'a, which corresponds to the Arena nodes are allocated in. Child Asts are wrapped in RefCell for interior mutability.
